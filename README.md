# 广州港数据采集监控系统 (Guangzhou Port Data Acquisition and Monitoring System)

这是一个基于 Web 的现代化港口数据采集与监控系统演示项目，旨在展示高韧性港口运营、绿色低碳转型以及数字化管理的通过能力。本项目采用单页应用 (SPA) 架构，结合 HTML5, CSS3 和 原生 JavaScript 开发，并通过 Chart.js 和 Three.js 实现丰富的数据可视化和数字孪生效果。

## ✨ 项目亮点

*   **实时运营监控**: 动态展示在港船舶、今日吞吐量、设备在线率等关键绩效指标 (KPI)，并提供实时图表分析。
*   **数字孪生可视化**: 利用 Three.js 构建 3D 港口场景，模拟码头泊位、起重机等设施的运行状态，支持视角控制和动画演示。
*   **绿色低碳管理**: 专门的碳排放管理模块，追踪碳排放总量、减排成果及碳中和进度，展示岸电、光伏等绿色措施的实施情况。
*   **应急指挥中心**: 集成实时告警系统、应急预案管理和应急资源调度功能，保障港口安全运营。
*   **现代化 UI 设计**: 采用响应式设计，结合流光效果、玻璃拟态等现代设计元素，提供优质的用户体验。

## 🛠️ 技术栈

本项目坚持轻量化和高性能的原则，不依赖庞大的前端框架：

*   **核心**: HTML5, CSS3, Vanilla JavaScript (ES6+)
*   **数据可视化**: [Chart.js](https://www.chartjs.org/) (用于统计图表)
*   **3D 渲染**: [Three.js](https://threejs.org/) (用于数字孪生场景)
*   **字体**: [Google Fonts](https://fonts.google.com/) (Inter 字体)

## 🚀 快速开始

由于本项目纯静态开发，无需复杂的构建过程：

1.  **直接运行**: 双击打开 `index.html` 文件即可在浏览器中预览。
2.  **本地服务 (推荐)**: 为获得最佳体验（避免跨域限制），建议使用 Live Server 或类似工具启动本地服务器。
    *   使用 VS Code: 安装 "Live Server" 插件，右键 `index.html` 选择 "Open with Live Server"。
    *   使用 Python: 在项目根目录运行 `python -m http.server`。
3.  **浏览器支持**: 建议使用最新版本的 Chrome, Edge 或 Firefox 浏览器。

## 🧩 功能模块详情

### 1. 实时监控 (Real-time Monitoring)
*   显示关键运营数据卡片（带趋势指示）。
*   可视化吞吐量趋势和设备状态。
*   实时更新的在港船舶动态表格。

### 2. 数字孪生 (Digital Twin)
*   交互式 3D 场景展示。
*   支持相机视角重置和动画开关。
*   侧边栏显示当前场景中的设备详情。

### 3. 碳排管理 (Carbon Management)
*   监控碳排放 KPI 与年度目标对比。
*   可视化排放源分布和历史趋势。
*   展示岸电、电动设备、光伏等减排措施状态。

### 4. 应急指挥 (Emergency Command)
*   实时告警滚动列表。
*   应急预案快速查看（火灾、台风、危化品）。
*   应急资源（车辆、人员、物资）状态监控。
